# Data Directory

## Overview
This directory stores operational metrics and usage data. **All files in this directory are gitignored** (not committed to the repository) as they contain sensitive employee or company information.

## AI Usage Data (Local Only)

### File: ai_usage_data.csv
- **Status**: **GITIGNORED** - Contains sensitive employee information (names, job titles, usage data)
- **Location**: Keep this file locally, do NOT commit to git
- **Update Frequency**: Weekly (as needed)
- **Purpose**: Tracks Claude and Devin usage for team users

### Usage Pattern

**To generate the AI Usage Report:**
```bash
# Copy your latest usage data to data/ai_usage_data.csv (locally)
# Then run the report with --file argument:
python execution/reports/usage_tables_report.py --file data/ai_usage_data.csv --open
```

**Example CSV Format:**
See `data/ai_usage_data_example.csv` for the required column structure. Your CSV must contain:
- `Name` - User name
- `Job Title` - User's job title
- `Software Company` - Company/division identifier
- `Claude Access?` or `Claude Access ` - Access status (Yes/No)
- `Claude 30 day usage` - Usage count for last 30 days
- `Devin Access?` or `Devin Access ` - Access status (Yes/No)
- `Devin_30d` - Devin usage count for last 30 days

## Other Data Files

This directory also contains:
- `baseline_*.json` - Project baseline metrics (gitignored)
- `weekly_tracking_*.json` - Weekly tracking data (gitignored)
- `armorcode_baseline.json` - Security baseline (gitignored)

All JSON files are generated by collectors and dashboards and should remain local only.

## Security Note
⚠️ **NEVER commit actual data files to git.** Only example files (`*_example.*`) should be tracked for documentation purposes.
